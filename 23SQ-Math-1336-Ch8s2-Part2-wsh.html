Inﬁnite Series Convergence Bathtub Analogy:

When thinking about convergence -vs- divergence of inﬁnite series, we can draw an analogy between ﬁlling up a bathtub with successive amounts of water according to some speciﬁc pattern:

∑ n=1∞a n = a1 + a2 + a3 + a4 + a5 + …

If the bathtub doesn’t overﬂow, that must mean that the amount of water we are adding each time goes to zero:


Figure 1: Illustration of the convergent series bathtub analogy for Theorem 8.2.6

Theorem 8.2.6:

If the series ∑ n=1∞a n is convergent, then limn→∞an = 0.

If the amount of water we add each time doesn’t go to zero, then the bathtub will deﬁnitely overﬂow:


Figure 2: Illustration of the bathtub analogy for the Test for Divergence.

Test for Divergence:

If limn→∞an DNE -OR- limn→∞an≠0, then ∑ n=1∞a n is divergent.
 
 

Theorem 8.2.8:

If ∑ an and ∑ bn are convergent series, then so are:

(i)
∑ can = c∑ an
(ii)
∑ (an + bn) = ∑ an + ∑ bn
(iii)
∑ (an − bn) = ∑ an −∑ bn

Series whose Convergence Behavior we Know:

Geometric Series:

∑ n=1∞arn−1 = a + ar + ar2 + ar3 + … + arn−1 + …

converges to S = a 1 − r if |r| < 1
diverges if |r|≥ 1.

Harmonic Series:

∑ n=1∞1 n = 1 + 1 2 + 1 3 + 1 4 + 1 5 + …

diverges

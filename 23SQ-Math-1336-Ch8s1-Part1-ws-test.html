Section 8.1, Part 1: Intro to Sequences


A sequence is an ordered list of numbers. An inﬁnite sequence is a sequence that does not terminate.

a1,a2,a3,…,an−1,an,an+1,…

Convergence/Divergence of Inﬁnite Sequences:


Figure 1: Sequence Illustrations: First graph shows a sequence that converges, second graph shows a sequence that diverges due to oscillation.

  A sequence {an} has the limit L:

limn→∞an = L,

if we can make the terms an “as close to L as we want” for n suﬃciently large.
 
If limn→∞an = L exists then we say that the sequence converges to L.
 
If limn→∞an does not exist then we say that the sequence diverges.
 

𝜖 − N Limit Deﬁnition:

The sequence an has the limit L if for every 𝜖 > 0 there is a corresponding integer N such that

if n > N, then |an − L| < 𝜖.

The following theorems must be formally cited when used:

Squeeze Theorem:

If an ≤ bn ≤ cn for n ≥ n0 and

limn→∞an = limn→∞cn = L,

then limn→∞bn = L.

Continuity & Convergence Theorem:
If limn→∞an = L and f is continuous at L, then limn→∞f(an) = f(L).

Theorem 3: If limx→∞f(x) = L and f(n) = an when n is an integer, then limn→∞an = L.

If you want to use L’Hopital’s Rule:

First switch to a function, f, of the continuous variable x, such that f(n) = an when n is an integer.
Then use L’Hopital’s Rule to ﬁnd the limit of the continuous function f(x) as x →∞.
Finally, cite Theorem 3.

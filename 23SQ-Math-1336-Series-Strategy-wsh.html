Summary of Convergence Tests for Series
Math 1336 & 1337, Spring 2023, Dr. Cole

Test

Series

Convergence or Divergence

Comments

Test for Divergence

∑ an

Diverges if limn→∞an≠0 (or DNE)

Inconclusive if limn→∞an = 0. More work must be done!

Geometric Series

∑ n=1∞arn−1

Converges to a 1 − r if |r| < 1.
Diverges if |r|≥ 1.

Useful for comparison tests if the nth term of an is similar to rn.

p−Series

∑ n=1∞ 1 np

Converges if p > 1.
Diverges if p ≤ 1.

Useful for comparison tests if the nth term an is similar to 1 np.

Integral

∑ n=c∞a n,c > 0
an = f(n)

Converges if ∫ c∞f(x)dx Converges.

Diverges if ∫ c∞f(x)dx Diverges.

The function f must be
positive, decreasing, & you must be able to integrate f(x) for x ≥ c.

Comparison

∑ an and ∑ bn
0 ≤ an ≤ bn

∑ bn Converges
    ⇒∑ an Converges.
∑ an Diverges ⇒∑ bn Diverges.

The comparison series is
often a geometric or p-series, (or a series to which another test may be applied).

Limit Comparison

∑ an and ∑ bn
an,bn > 0,
limn→∞an bn = c > 0

∑ bn Converges
    ⇒∑ an Converges.
∑ bn Diverges ⇒∑ an Diverges.

To ﬁnd bn consider only the terms of an that have the greatest eﬀect when n is large.

Alternating Series

∑ n=1∞(−1)n−1b n,

bn > 0

Converges if
   (i): bn+1 ≤ bn
and (ii): limn→∞bn = 0

Applicable only to series with alternating terms. (Note that (−1)n−1 is not the only quantity that can cause alternation. Carefully consider series including cos(nπ), etc.)

Absolute Convergence

∑ an

∑ |an| Converges
    ⇒∑ an Converges.

Useful for series containing both positive and negative terms. You may apply other tests to ∑ |an| since all of the terms will be positive.

Ratio

∑ an
limn→∞|an+1| |an| = L

Converges (absolutely) if L < 1.
Diverges if L > 1 or L = ∞.

Inconclusive if L = 1.
Test fails, try another test! Useful if an contains terms like: n!, np, or cn.

Root

∑ an
limn→∞|an|n = L

Converges (absolutely) if L < 1.
Diverges if L > 1 or L = ∞.

Inconclusive if L = 1.
Test fails, try another test! Useful if an contains
nth powers.
